### Projects

**[hallucination-detector](https://github.com/JKDasondee/hallucination-detector)** — LLM hallucination detection pipeline. Claim extraction via spaCy NER, evidence retrieval with ChromaDB RAG, NLI scoring via DeBERTa cross-encoder. F1 96.8% on 30-item benchmark. `pip install hallucination-detector`

**[av-stoikov-mm](https://github.com/JKDasondee/av-stoikov-mm)** — Avellaneda-Stoikov market maker on Bybit BTCUSDT L2 tick data via hftbacktest. Reservation price, optimal spread, queue position model.

**[strategy-predictor](https://github.com/JKDasondee/strategy-predictor)** — PyTorch LSTM predicting DeFi strategy performance. Trained on real Glider.Fi protocol data. Go REST API serving predictions.

**[wbc](https://github.com/JKDasondee/wbc)** — On-chain wallet behavior classifier. K-means clustering on EVM transaction patterns, written in Go.

**[community-signal-engine](https://github.com/JKDasondee/community-signal-engine)** — Real-time community portfolio intelligence for DeFi. Activity scoring, strategy ranking, Discord integration.
